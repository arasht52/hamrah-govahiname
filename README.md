# همراه گواهینامه 🇩🇪🚗

اپلیکیشن آموزشی فارسی برای آمادگی آزمون گواهینامه رانندگی آلمان.

## امکانات MVP

- تمرین سوالات گواهینامه
- توضیح فارسی ساده
- تحلیل نتیجه آزمون
- دستیار آموزشی Demo Mode
- ساختار آماده برای افزودن سوالات بیشتر

## تکنولوژی

- React
- Vite
- JavaScript
- LocalStorage
